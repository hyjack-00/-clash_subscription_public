# Clash Meta for Android

Clash Meta for Android is a rule-based network toolbox built around the Mihomo core.

## Features

- profile management
- rule-based routing
- multiple outbound protocols
- DNS and TUN integration
- Android-oriented workflow

## Project Layout

- `app/`
- `common/`
- `docs/`

## Build

This repository uses a standard Android project layout. Open it with Android Studio or build with Gradle in a compatible environment.
